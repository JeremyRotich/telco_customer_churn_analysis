# Telco Customer Churn Analysis

This project analyzes customer churn behavior in a telecommunications dataset using SQL Server and Power BI.  
The goal is to identify factors that influence customer churn and support data-driven retention strategies.

---

## Business Objective
To understand why customers leave and identify patterns that can help improve customer retention.

---

## Tools Used
- SQL Server (data cleaning and analysis)
- Power BI (dashboard and visualization)

---

## Key Insights
- Customers on month-to-month contracts show higher churn rates
- Lack of additional services increases churn likelihood
- Contract type is a strong predictor of customer retention

---

## Project Files
- SQL scripts for data cleaning and analysis
- Power BI dashboard file (.pbix)
- Dashboard screenshots included in /images folder
